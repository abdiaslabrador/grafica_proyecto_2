a) In case of run the .exe program without compiling.
- Run the .exe.

b) In case of compiling and executing the program in windows.
- Just run the file that says compiler.cmd, this will open a (terminal), wait for the cmd to close (this process compiles the program).
- Once the cmd is closed, execute the main.exe that has just been generated.

c) In case of compiling and executing the program in linux.
- Open the terminal, go to the project address and run this command to compile the main.cpp "g++ -o main main.cpp -I gladinclude -lglfw -lGL"
- Once the program is compiled, run the generated executable.
